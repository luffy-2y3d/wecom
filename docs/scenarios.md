### 💬 收发消息

允许机器人获取单聊、群聊消息，并可向指定用户和群聊发送消息。消息总结、待办汇总、通知提醒场景更便捷；（收发单聊和群聊消息）

### 功能特点

- **智能总结**：自动分析聊天内容，提取关键信息并生成结构化总结
- **多群通知**：支持一次性向多个群聊发送相同通知，提高工作效率
- **实时响应**：消息处理和发送速度快，确保信息及时传递
- **结构化展示**：采用表格等结构化方式展示信息，清晰易读

### 配置建议

为了充分发挥消息收发功能，建议在 `openclaw.json` 中进行以下配置：

```json
{
  "channels": {
    "wecom": {
      "enabled": true,
      "defaultAccount": "default",
      "accounts": {
        "default": {
          "enabled": true,
          "bot": {
            "primaryTransport": "ws",
            "ws": {
              "botId": "YOUR_BOT_ID",
              "secret": "YOUR_BOT_SECRET"
            }
          },
          "agent": {
            "enabled": true,
            "corpId": "YOUR_CORP_ID",
            "agentId": 1000001,
            "agentSecret": "YOUR_AGENT_SECRET"
          }
        }
      }
    }
  }
}
```

### 最佳实践

1. **明确指令**：在请求中明确时间范围、对象和任务类型，如"帮我总结昨天的群聊内容"
2. **指定群聊**：发送通知时，清晰列出群聊名称，确保机器人能准确识别目标群聊
3. **合理使用**：将机器人用于提高工作效率的场景，如会议通知、工作总结等
4. **定期更新**：根据业务需求，及时调整机器人的配置和功能
